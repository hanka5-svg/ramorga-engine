# 04_meniscus_engine

Executable implementation of the Homeostatic Meniscus:
- regulatory functions,
- modulation rules,
- invariant enforcement.
- [Field Archetypes — HFS](../field_archetypes.md)  
  Archetyp pola używany do walidacji stabilności menisku przy interferencji wysokiej gęstości.

MeniscusEngine działa wyłącznie na poziomie modulacji amplitudy interferencji i nie ingeruje w geometrię pola.
