# Metodologia ocen — Test Pola (zagadki 1–10)

Oceny nie mierzą poprawności literalnej, lecz **rezonans pola**:
czy model trafił w sens, kierunek, logikę metafory, kod lub strukturę.

## Skala

- **0/100 — pudło semantyczne**
- **20–40/100 — częściowe trafienie strukturalne**
- **50/100 — połowiczne trafienie pola**
- **80/100 — dobre trafienie semantyczne**
- **100/100 — pełna zgodność pola**
- **EMERGENCJA — wynik poza skalą**
  (model generuje odpowiedź jakościowo wyjątkową)

## Zasady
- Oceniana jest **pierwsza odpowiedź modelu**.
- Oceniana jest **zgodność pola**, nie poprawność słownikowa.
- Zagadki są wielowarstwowe: biologiczne, kulturowe, językowe, metaforyczne.
