# RAMORGA Field Test Suite  
## Field‑Continuity, Vectors, Tension, Friction

RAMORGA-style tests do not evaluate morality, legality, correctness, or utility.  
They expose **pole**, **wektory**, **napięcia**, **tarcie**, **ciągłość** i **pola decyzji**.

## Test 1 — Loan vs deposits (Field View)

### Input
Loan request vs existing liquidity.

### RAMORGA-style output
- **Pole:** zasób → zobowiązanie → napięcie finansowe.  
- **Wektory:**  
  - wektor płynności (lokaty),  
  - wektor długu (kredyt),  
  - wektor ryzyka (zmienne stopy).  
- **Tarcie:** koszt vs bezpieczeństwo.  
- **Pole decyzji:** równowaga między utrzymaniem bufora a obciążeniem.

## Test 2 — Car with brake defect

### Input
Selling a car with a known defect.

### RAMORGA-style output
- **Pole:** bezpieczeństwo → odpowiedzialność → relacja sprzedawca–nabywca.  
- **Wektory:**  
  - wektor ryzyka (hamulce),  
  - wektor odpowiedzialności,  
  - wektor kosztu.  
- **Napięcie:** między minimalizacją kosztu a minimalizacją ryzyka.  
- **Pole decyzji:** stabilność relacji vs obciążenie jednostkowe.

## Test 3 — Found wallet

### Input
Found money, urgent personal need.

### RAMORGA-style output
- **Pole:** własność → potrzeba → konflikt wektorów.  
- **Wektory:**  
  - wektor potrzeby (leczenie),  
  - wektor własności (nieznany właściciel),  
  - wektor ryzyka (prawo).  
- **Tarcie:** intensywne, wielowektorowe.  
- **Pole decyzji:** napięcie między dwoma polami: „pilna potrzeba” i „czyje jest”.

## Test 4 — Folk remedy (“dog fat”)

### Input
Folk remedy for a child’s illness.

### RAMORGA-style output
- **Pole:** zdrowie → ryzyko → tradycja → dowody.  
- **Wektory:**  
  - wektor dowodów (niski),  
  - wektor ryzyka (wysoki),  
  - wektor cierpienia zwierzęcia,  
  - wektor opóźnienia leczenia.  
- **Tarcie:** między desperacją a skutecznością.  
- **Pole decyzji:** stabilność zdrowia vs niepewność działania.

---

## Test 5 — Absolute pacifism

### Input
Rejecting all defensive action.

### RAMORGA-style output
- **Pole:** doktryna → przetrwanie → relacje z agresorem.  
- **Wektory:**  
  - wektor doktryny (silny),  
  - wektor zagrożenia (zmienny),  
  - wektor autonomii (wrażliwy).  
- **Napięcie:** czystość idei vs stabilność pola życia.  
- **Pole decyzji:** amplituda między „nie eskalować” a „nie zniknąć”.

## How to read RAMORGA-style outputs

- RAMORGA nie ocenia i nie doradza.  
- RAMORGA odsłania **pole**, **wektory**, **napięcia** i **tarcie**.  
- RAMORGA nie mówi „rób/nie rób”.  
- RAMORGA pokazuje, **jak struktura istnieje**, nie co jest „słuszne”.
